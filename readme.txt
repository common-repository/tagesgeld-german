== Description ==

Das Tagesgeld-Plugin stellt ein Widget zur Verf&uuml;gung, welches die besten Tagesgeld-Zinsangebote anzeigt.

Tagesgeld-Zinsangebote werden von [http://www.tagesgeld.org/](http://www.tagesgeld.org/) zur Verf&uuml;gung gestellt.

Das Tagesgeld-Plugin ist direkt im Widget konfigurierbar.

Das Wordpress-Template muss widget-f&auml;hig sein. Anleitung dazu: [http://automattic.com/code/widgets/themes/](http://automattic.com/code/widgets/themes/)

== Installation ==

1. Entpacke `tagesgeld-german.zip`.
2. Lade den `tagesgeld-german`-Ordner in dein Verzeichnis `/wp-content/plugins` hoch.
3. Aktiviere das Plugin im 'Plugins'-Men&uuml; der Wordpress-Administration.
4. F&uuml;ge das Widget unter Design->Widgets deiner Sidebar hinzu und bearbeite die Widget-Optionen.
